### Own build/development

Dev:

-   npm

```BASH
$ npm i
$ npm run dev
```

-   pnpm

```BASH
$ pnpm i
$ pnpm dev
```

**Dev mode can be run with --host to run server on 0.0.0.0**

Build:

-   npm

```BASH
$ npm i
$ npm run build
$ npm run start
```

-   pnpm

```BASH
$ pnpm i
$ pnpm build
$ pnpm start
```
