# Installation
```BASH
$ python -m venv ./env
$ source ./env/bin/activate
$ pip install -r requirements.txt
```

# Django server
```BASH
$ ./manage.py runserver
``` 

# Node css + ts
```BASH
$ cd static
$ npm i # or pnpm i
$ npm run watchcss # or pnpm watchcss to build tailwindcss
$ npm run watchts # or pnpm watchts to transpile typescript files
```
